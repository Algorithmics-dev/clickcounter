# ⚡ Button Frenzy Arena ⚡

Καλώς ήρθες στο Button Frenzy Arena!

Σε αυτό το mini-game πρέπει να πατήσεις το κουμπί όσο πιο γρήγορα μπορείς πριν τελειώσει ο χρόνος.

Όσο περισσότερα clicks κάνεις:
- ξεκλειδώνεις νέα ranks
- παίρνεις πιο chaotic messages
- γίνεσαι Button Legend 👑

Το project είναι φτιαγμένο με Python + PyQt και μπορείς να το αλλάξεις όπως θέλεις.

# 🔥 Remix Ideas

## Easy Remixes
- άλλαξε τα χρώματα του interface
- άλλαξε το κείμενο του button
- άλλαξε τη διάρκεια του timer
- βάλε δικά σου emojis

## Creative Remixes
- φτιάξε δικό σου game theme
- κάνε sci-fi version
- κάνε meme version

## Chaos Mode Ideas
- random flashing colors
- ultra speed mode
- fake warning popups

## Challenge Mode
- μπορείς να φτάσεις 100 clicks;
- φτιάξε δικό σου νέο rank
- πρόσθεσε νέο game mechanic

## Showcase
Αν κάνεις δικό σου remix στείλε μας το παιχνίδι (screenshot) στο κάναλι **Showcase** και μπορεί να δεις όνομα σου στο leaderboard.
